# Check if a PDF is marked as PDF/A-compliant

This sample demonstrates how to read the declared PDF/A conformance level of a PDF document using the [Docotic.Pdf library](https://bitmiracle.com/pdf-library/).

## Description

Use the [ReadPdfaConformance](https://api.docotic.com/conformance/pdfdocumentconformanceextensions-readpdfaconformance) extension method
to read the document's declared PDF/A conformance level.

This sample code uses free [Conformance add-on](https://www.nuget.org/packages/BitMiracle.Docotic.Pdf.Conformance/) for the Docotic.Pdf library.

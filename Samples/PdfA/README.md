# PDF/A
This group contains samples that show how to produce PDF/A-compliant documents in C# and VB.NET using [Docotic.Pdf library](https://bitmiracle.com/pdf-library/).

Don't miss [all other samples](/Samples).

[Create PDF/A-compliant documents](/Samples/PdfA/CreatePdfA)

This sample shows how to create PDF/A-compliant documents using Docotic.Pdf library.

[Convert PDF to PDF/A](/Samples/PdfA/ConvertPdfToPdfA)

This sample demonstrates how to convert PDF documents to PDF/A-compliant files.

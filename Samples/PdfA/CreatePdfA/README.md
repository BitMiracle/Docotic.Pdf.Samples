# Create PDF/A-compliant documents in C# and VB.NET

This sample shows how to create a PDF/A document using [Docotic.Pdf library](https://bitmiracle.com/pdf-library/).

## Description

The free [Conformance add-on](https://www.nuget.org/packages/BitMiracle.Docotic.Pdf.Conformance/) provides
[SaveAsPdfa](https://api.docotic.com/conformance/pdfdocumentconformanceextensions-saveaspdfa) extension methods for `PdfDocument`.
You can use them to produce documents in the following PDF/A conformance levels:
* PDF/A-1a, PDF/A-1b
* PDF/A-2a, PDF/A-2b, PDF/A-2u
* PDF/A-3a, PDF/A-3b, PDF/A-3u
* PDF/A-4, PDF/A-4e, PDF/A-4f

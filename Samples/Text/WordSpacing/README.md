# Change word spacing on a PDF canvas in C# and VB.NET

This sample shows how to change word spacing using [Docotic.Pdf library](https://bitmiracle.com/pdf-library/).

## Description

Word spacing defines a distance between adjacent words in a line of text. Use the [PdfCanvas.WordSpacing](https://api.docotic.com/pdfcanvas-wordspacing) property to increase or reduce the spacing between words.
